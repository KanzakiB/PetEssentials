# Happy Tails

An e-commerce platform dedicated to dogs and cats, offering premium pet food, accessories, grooming supplies, toys, vitamins, and health essentials.


## Project Status

**Last Active:** 2024

## About The Project

Happy Tails is an e-commerce platform designed for pet lovers, offering a seamless shopping experience with secure payments, easy navigation, and fast delivery to keep your furry friends happy and healthy. The website aims to provide a one-stop shop for all pet essentials including food, toys, grooming supplies, accessories, vitamins, and health products for dogs and cats.


## Key Features

### User Side

- **Product Browsing** – View products by category
- **Shopping Cart** – Add and manage items
- **User Registration** – Create an account
- **User Login** – Secure access to account
- **Order Management** – View order history

### Admin Side

- **Admin Dashboard** – View statistics and manage products
- **Product Management** – Add, edit, or delete products
- **Category Management** – Manage product categories
- **User Management** – View registered users

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- XAMPP ([Download here](https://www.apachefriends.org/))
- Visual Studio Code ([Download here](https://code.visualstudio.com/))
- Web browser


## Installation

Step 1: Install and run XAMPP
Start Apache and MySQL in the XAMPP Control Panel.

Step 2: Create the database
Go to localhost/phpmyadmin and create a new database named **petessentialsdb**.

Step 3: Import the SQL file
In phpMyAdmin, select your database, click the Import tab, choose the SQL file, and click Go.

Step 4: Move project files
Copy the project folder to C:\xampp\htdocs\

Step 5: Open in VSCode
Open Visual Studio Code, select File > Open Folder, and choose your project folder.
